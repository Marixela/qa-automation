Collections of json files to test Todoist API for Wizeline QA Certification

For this task we used Todoist API (https://developer.todoist.com/rest/v8/#overview), which is the public API for the Todoist app.

Endpoints Validated:

Get all projects
Create a new Project
Get a Project
Update Project
Delete Project
Get active Tasks
Create a new Task
Get a Task
Update a task
Close a task
Reopen a task
Delete Task
